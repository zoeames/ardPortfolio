# ARD Portfolio
#### Description:
Art portfolio website.

####To run:
```
$ npm install
$ gulp clean
$ gulp build
$ node index.js
```

Your app should now be running on [localhost:5001](http://localhost:5001/).

####When developing run:
```
$ npm install
$ node index.js
$ gulp watch
```


#### Deploying to Heroku
```
$ heroku create
$ git push heroku master
$ heroku open
```
